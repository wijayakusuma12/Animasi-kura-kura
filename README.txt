Kura-kura Interaktif dengan Suara Langkah (HTML + Canvas + Web Audio API)
==========================================================================

Isi file:
- index.html : Halaman utama (animasi kura-kura, kontrol keyboard & touch, suara langkah "boop-boop" digenerate via Web Audio API)
- README.txt  : File ini

Cara pakai:
1. Ekstrak folder dari ZIP.
2. Buka file `index.html` dengan browser (Chrome/Firefox/Edge/Safari).
3. Tekan tombol panah (← ↑ → ↓) atau WASD untuk menggerakkan kura-kura.
4. Di perangkat layar sentuh, ketuk area tampilan lalu geser untuk mengontrol arah.
5. Klik area tampilan pertama kali jika suara tidak berbunyi — beberapa browser meminta interaksi pengguna dulu sebelum mengizinkan audio.

Catatan:
- Suara langkah dihasilkan langsung lewat Web Audio API (tidak perlu file audio eksternal).
- Jika audio tidak berfungsi di browser, pastikan tab tidak dibisukan dan berikan izin untuk memutar suara (biasanya dengan klik/interaksi).
